# FrostPeak_Survival

설산에서 체온을 관리하며 정상에 도달해야하는 2D 생존 게임

## 프로젝트 개요

- 장르: 2D 생존 게임
- 개발 엔진: Unity 6
- 개발 언어: C#
- 개발 기간: 1주 압축 개발
- 핵심 목표: 제한된 시간 안에 완성 가능한 생존 게임 MVP 구현

## 게임 목표

플레이어는 설산에서 이동하며 체온을 유지한 채로 설산 정상까지 가야합니다.  
체온이 0이 되면 게임오버가 되며, 모닥불과 베이스 캠프를 활용하여 정상에 도달하십시오. 

## 핵심 시스템

### 플레이어 이동
- WASD 이동
- Rigidbody2D 기반 이동
- 걷기 애니메이션 연동
- 좌우 방향 전환

### 체온 시스템
- 시간이 지날수록 체온 감소
- 체온이 0이 되면 게임오버
- 안전지대(베이스 캠프, 모닥불)에서 체온 회복

### UI
- 현재 체온 표시
- 생존 시간 표시
- 게임오버 화면 예정

## 개발 과정

### 1. 기획 축소
초기에는 인벤토리, 제작, 채집, NPC 등 다양한 생존 요소를 고려했지만,  
제한된 개발 기간을 고려하여 핵심 경험을 “설산 속 체온 관리 및 정상 도달”로 압축했습니다.

### 2. 스프라이트 애니메이션 적용
플레이어 걷기 스프라이트 시트를 Unity에 적용하고 Slice 작업을 진행했습니다.

### 3. 애니메이션 정렬 문제 해결
애니메이션 재생 중 캐릭터가 좌우로 밀리는 문제가 발생했습니다.  
원인을 확인한 결과, 각 프레임의 캐릭터 위치가 동일한 기준점에 정렬되어 있지 않았습니다.  
프레임을 직접 수정하여 캐릭터가 제자리에서 자연스럽게 걷도록 개선했습니다.

## 트러블슈팅

### 문제1: 걷기 애니메이션 재생 시 캐릭터가 좌우로 밀림

원인:
- Sprite Sheet의 각 프레임 내부에서 캐릭터 위치가 서로 달랐음
- Unity의 Pivot 설정만으로는 완전히 해결되지 않음

해결:
- 각 프레임을 직접 수정
- 캐릭터의 기준 위치를 맞춤
- 애니메이션이 제자리에서 자연스럽게 재생되도록 개선
